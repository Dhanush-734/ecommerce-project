# 📊 E-commerce Analytics Dashboard

A full-stack e-commerce backend with an analytics dashboard built using FastAPI and Streamlit.

## 🚀 Features

- 🔐 JWT Authentication (Login/Register)
- 🛍️ Product Management
- 🛒 Cart System
- 🧾 Order Processing (Checkout)
- 📊 Real-time Analytics Dashboard
- 💰 Revenue & Order Tracking
- 📈 Data Visualization (Trends & Insights)

## 🛠️ Tech Stack

- Backend: FastAPI
- Database: MySQL / SQLite
- ORM: SQLAlchemy
- Authentication: JWT (python-jose)
- Dashboard: Streamlit
- Data Handling: Pandas

## 📊 Dashboard Insights

- Total Orders
- Total Revenue
- Orders Over Time
- Revenue Trends
- Product Stock Visualization

## 🔄 Workflow

1. User registers/logs in
2. Products are added
3. User adds items to cart
4. Checkout creates orders
5. Dashboard visualizes business insights

## 🌐 Deployment (Planned)

- Backend: Render
- Dashboard: Streamlit Cloud

## 🎯 Purpose

This project demonstrates backend development, database design, and data analytics using real-world e-commerce workflows.
